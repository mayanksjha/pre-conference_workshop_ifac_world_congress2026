IFAC 2026 WORKSHOP MICROSITE
============================

This folder contains a GitHub Pages-ready static website for the proposed IFAC 2026 pre-congress workshop:

Safe, Fault-Resilient & Health-Aware Control Design and Learning

Files
-----
- index.html        Main one-page website
- styles.css        Green-themed stylesheet
- assets/images/    Supplied organizer/participant images

How to publish on GitHub Pages
------------------------------
1. Create a GitHub repository.
2. Upload the full contents of this folder to the repository root.
3. In GitHub: Settings -> Pages.
4. Under "Build and deployment", choose:
   - Source: Deploy from a branch
   - Branch: main (or master)
   - Folder: /(root)
5. Save. GitHub will publish the site in a minute or two.

Quick edit tips
---------------
- Open index.html in any text editor.
- Search for the phrase "Provisional talk theme" to update talk titles and descriptions.
- Search for "Suggested scientific contact" to change the contact person.
- Search for "Workshop abstract" and "Illustrative workshop flow" to replace draft text.
